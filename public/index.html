Chris Duke

Web Developer

My name is Chris Duke. I'm a web developer and graduate of The Iron Yard Nashville, an immersive code education program. My area of concentration was Front-End Engineering, with a focus on HTML, CSS, JavaScript, and Ember.JS. After building a couple of projects using the Adonis.JS framework I've become increasingly interested in back end development. I find immense joy in problem solving, I look forward to continuing to learn new skills I can apply to both client and server side programming. In my spare time you can usually find me with guitar in hand, transcribing arrangements for classical guitar or collaborating with my friends on musical creations.
